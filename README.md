# Akram Fish House 🐟

Welcome to the official website repository for **Akram Fish House** – a premium seafood restaurant offering the freshest and most delicious fish dishes. This project is a fully responsive, beautifully designed front-end web application built to enhance the online presence of the restaurant.

## 🚀 Features

- **Modern & Responsive Design:** Flawless experience across all devices (Desktop, Tablet, Mobile).
- **Interactive Menu:** Beautifully categorized menu sections for easy browsing.
- **Engaging UI/UX:** Smooth animations, hover effects, and a premium color palette.
- **Contact & Location:** Integrated contact form and location details for customer convenience.

## 💻 Tech Stack & Tools

**Languages / Technologies:**

- **HTML5:** Structure & content of the website.
- **CSS3:** Custom styling, flexbox, grid, and animations.
- **JavaScript:** DOM manipulation, handling dynamic menu data (`js/menu-data.js`), and interactions.

**Tools Used:**

- **Visual Studio Code (VS Code):** Primary code editor for development.
- **Git & GitHub:** Version control and source code hosting.
- **Local HTTP Server:** For testing the site locally.

## 🛠️ How to Edit / کیسے ایڈٹ کریں؟

### English Instructions

To edit this website, use a text editor like **Visual Studio Code (VS Code)**:

1. **Pages (`index.html`, `menu.html`, `contact.html`, `about.html`):** Edit these HTML files to change the text content or layout details.
2. **Styling (`css/style.css`):** Modify this file to change colors, fonts, or margins.
3. **Menu Items (`js/menu-data.js`):** To add, remove, or modify the menu dishes and prices, edit the array in this file without needing to change the HTML directly!

### اردو ہدایات (Urdu Instructions)

اس ویب سائٹ کو ایڈٹ کرنے کے لیے **Visual Studio Code (VS Code)** کا استعمال کریں:

1. **ویب پیجز (`index.html`, `menu.html` وغیرہ):** ویب سائٹ پر لکھی ہوئی تحریر تبدیل کرنے کے لیے ان HTML فائلوں میں ایڈیٹنگ کریں۔
2. **ڈیزائن اور رنگ (`css/style.css`):** ویب سائٹ کے رنگ، فاؤنٹ یا ڈیزائن بدلنے کے لیے CSS فائل میں تبدیلی کریں۔
3. **مینو کی لسٹ (`js/menu-data.js`):** اگر آپ نے مینو میں نئی ڈشز کا اضافہ کرنا ہے، قیمت بدلنی ہے یا کوئی ڈش ہٹانی ہے تو اس فائل میں موجود لسٹ کو ایڈیٹ کریں۔ اس طرح آپ کو HTML میں تبدیلی نہیں کرنی پڑے گی!
